## 📖 Descrição

O **DoarJuntos** é uma aplicação web desenvolvida para facilitar a conexão entre pessoas que desejam doar itens e instituições ou pontos de coleta que precisam receber essas doações.

O projeto tem como objetivo social:

- **Incentivar a solidariedade** por meio de uma plataforma simples e acessível.
- **Mapear pontos de coleta** para que doadores encontrem locais próximos.
- **Promover a transparência** nas doações, conectando quem doa e quem recebe.

## ⚙️ Funcionalidades Baseadas no Modelo 3C

O Modelo 3C de groupware auxilia no entendimento de como o sistema suporta a interação entre pessoas. No DoarJuntos, as funcionalidades são organizadas da seguinte forma:

### 💬 Comunicação

- Formulário de contato entre doador e ponto de coleta.
- Mensagens de confirmação e feedback após uma doação.
- Notificações sobre novos pontos de coleta cadastrados.

### 🗓️ Coordenação

- Calendário de campanhas e eventos de arrecadação.
- Organização das doações por categoria e localidade.
- Status da doação (pendente, em andamento, concluída).

### 🤝 Cooperação

- Cadastro colaborativo de pontos de coleta.
- Histórico compartilhado de doações realizadas.
- Integração entre doadores, instituições e comunidade.

## 🚀 Como Executar o Projeto

Siga os passos abaixo para executar o DoarJuntos localmente utilizando o **Live Server** no Visual Studio Code:

1. **Clone ou baixe o repositório** do projeto para o seu computador.
2. **Abra a pasta do projeto** no Visual Studio Code.
3. **Instale a extensão Live Server** caso ainda não tenha:
   - Clique no ícone de extensões (`Ctrl + Shift + X`).
   - Pesquise por **Live Server** e instale a extensão de Ritwick Dey.
4. **Clique com o botão direito no arquivo `index.html`**.
5. Selecione a opção **"Open with Live Server"**.
6. O navegador padrão será aberto automaticamente com o projeto em execução.

> 💡 Dica: certifique-se de que o arquivo `index.html` está na raiz do projeto para que o Live Server o reconheça como página inicial.
